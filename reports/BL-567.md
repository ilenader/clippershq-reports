## PART 4 — verdict
The ~$0.20/hour is buying, in this window, ~$0.17/hour of DELETED-reel lookups (a leak) and ~$0.03/hour
of genuine floor (real IG rescues + the TikTok LamaTok floor). A gone path survived.

**Root cause (two defects, both Instagram):**
1. Gone reels are NEVER retired. tracking.ts classifies a gone/null return as INFRA_DEFER (no strike,
   consecutiveFailures stays 0), so a deleted reel never reaches a terminal state and keeps polling at
   ~28h forever. Nothing sets videoUnavailable=true from repeated definitive-gone verdicts.
2. The only thing sparing Apify each tick is the per-tick gone-skip (apify.ts:2155), which fails during
   any HikerAPI cooldown/rate burst; then all due gone reels escalate to the gone-blind
   fetchInstagramStats (apify.ts:616) at $0.003 + $0.005 each, both SUCCEEDED-but-empty.

**Fix (durable, one place):** split the DEFINITIVE-gone return from the transient miss in tracking.ts
(mirroring BL-559's "definitive not-found" distinction). On the Kth consecutive HikerAPI verdict="gone"
(404 MediaNotFound), set `clip.videoUnavailable=true` + `videoUnavailableSince` AND retire the job —
either `trackingJob.isActive=false` or add `clip: { videoUnavailable: false }` to the dueJobs where
clause at cron/tracking/route.ts:298 (today it filters only isActive + nextCheckAt, so marking the clip
alone will NOT stop polling). Retired clips leave the rotation, so the cooldown-burst escalation cannot
recur. This obeys the never-zero rule: videoUnavailable freezes earnings at last-known (savedEarnings),
it does not zero them, and tracking.ts:1720 auto-restores on recovery if the reel ever returns.
**Defensive add:** in fetchInstagramStats (apify.ts:616), a cheap by/code 404 check before the billed
tiers, so a cooldown can never route a gone reel to a billed actor even before retirement lands.

**Projected real cost (from the POST-fix run rate, not the day's cumulative):** at $0.20/hour sustained,
total Apify ~= **$4.8/day / ~$145/month**. Of that ~70 to 86% is the IG gone-leak = **~$100 to $125/month**
recoverable. The honest floor that must remain (real IG rescues + TikTok LamaTok miss) is **~$0.03/hour
= ~$15 to $20/month**; it cannot go lower without breaking clippers, because an unresolvable live clip
must return NULL (defer prior views), never 0.
